pilot status codes

overview

Here is an overview of the status codes that are current in-use. Note that the color-coding helps to clarify the type of message shown.

typical use

sent by pilots

  • FLY
    Pilot was checked out through gate and is flying.
  • LOK
    Pilot has Landed OK
  • PUP
    Pilot has been Picked UP (either by retrieve, another vehicle, or is otherwise where they want to be now)
  • AID
    Pilot requires assistance (but issue is not life-threatening). If life-threatening, press your "SOS" button and dial 911 if you are able.

set by HQ

  • (none)
    At the start of each day the status for all pilots is set to (none) (a blank). If there are known other situations for a pilot then their status may be automatically set to something else (such as ABS).
  • DRA
    Driver/Vehicle A has been assigned to retrieve.

special codes

sent by pilots

  • MSG
    A pilot needs to communicate with HQ and has no ability to phone or call on the radio. HQ must locate the message details, so this is not an instaneous method of communication.
  • DNF
    Pilot checked out to fly on launch, but ultimately Did Not Fly.

set by HQ

  • ASK
    ASK at HQ to find out what is needed. There is likely a problem with your tracker configuration or message format.
  • ABS
    A pilot known to be absent; for example, if they have withdrawn.
  • GOL
    A pilot landed in Goal, but not yet confirmed in a vehicle (PUP).
  • SPOT
    Pilot using a foreign SPOT device, where messages are sent via email instead of SMS. These messages can not be automatically processed - someone in HQ must read them and manually set the pilot's status.
  • FIN
    Everything has been verified and you are FINished with your Bowerbird set up.

set by bowerbird

  • (random)
    Any message with a valid pilot number but unrecognized status information.
  • DNF
    Pilot checked out to fly on launch, but ultimately Did Not Fly.
  • ERR
    A valid pilot number was provided, but there was an issue with the rest of the message.
NOTE: Additional status codes may be created during the competition, to accomodate new situations.
